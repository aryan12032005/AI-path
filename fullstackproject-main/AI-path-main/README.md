# AI Career Compass

A modern web application designed to help users navigate their AI career paths with personalized guidance and resources.

## Features

- **Career Path Exploration**: Discover various AI-related career opportunities
- **Personalized Recommendations**: Get tailored suggestions based on your interests and skills
- **Resource Library**: Access curated learning materials and guides
- **Interactive Components**: Engage with modern, responsive UI components
- **Contact & Support**: Get in touch with mentors and career advisors

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or bun package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Asmitt02/AI-career-path.git

# Navigate to the project directory
cd ai-career-compass

# Install dependencies
npm install
# or
bun install
```

### Development

```bash
# Start the development server
npm run dev
# or
bun dev
```

The application will be available at `http://localhost:5173`

### Building

```bash
# Build for production
npm run build
# or
bun build
```

## Project Structure

```
src/
├── components/        # React components
├── pages/            # Page components
├── hooks/            # Custom React hooks
├── lib/              # Utility functions
├── assets/           # Static assets
├── App.tsx           # Main app component
└── main.tsx          # Entry point
```

## Technologies Used

- **Frontend Framework**: React 18
- **Type Safety**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui with Radix UI
- **Routing**: React Router
- **Forms**: React Hook Form with Zod validation
- **Testing**: Vitest & Playwright
- **Build Tool**: Vite

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
- `npm run test` - Run tests once
- `npm run test:watch` - Run tests in watch mode
